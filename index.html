    Health insurance prediction model.

    Insurance policies have been common nowadays with different companies offering different policies against different risks. In this project I focused on creating a machine model that would aim to predict the amount of premium to be paid by individuals for a health policy. I utilized a kaggle data set which was relatively clean and performed different ETL processes and encoding of categorical variables. I later created a correlation matrix to measure the strength of the input features. I used the decision tree algorithm and also the random forest regression model to evaluate performance. I later performed the grid search to obtain appropriate parameters that would work relatively well with unseen data. I retrained the model with the new parameters and gave me an accuracy of about 87% showing that the model worked well.
